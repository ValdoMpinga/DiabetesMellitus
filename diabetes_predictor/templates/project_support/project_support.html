Contribuindo com seus dados

Muito obrigado por considerar em contribuir para o website, para contribuir para o website, somente precisa se registar no website usando o seu email e permitir que os dados dos seus diagnostico sejam armazenados no sistema do site.
O registro por email serve para garantir que uma pessoa real esta a contribuiir para o projeto com dados solidos e coerentes.

Ao contribuir esteja seguro que:

Um modelo mais preciso
O modelo estará um passo a mais de se tornar mais forte, mais preciso, pois um modelo de inteligencia artificial alimenta se com dados. E se o mesmo crescerá graças a si!
Segurança
Os dados do seu diagnostico serão armazenados de forma segura e anonima. Os mesmos jamais serão partilhados com terceiros!
Autenticação por email
O email é necessario para garantir que cada pessoa contribua para o modelo no maximo uma vez por ano, no entanto podem na mesma tirar quantos diagnostico quiserem!
